
# Simple Python calculator.

## Simple demo to show you my python skills.

This is just tier-1; more levels on the way.

It's a small calculator, you can download the code and run it on your machine.

Place the code in any folder of your choise; inside, open command line (ex. "cmd" in Windows) and type:

"python calc.py"

you will be prompted with the action you want to perform.
